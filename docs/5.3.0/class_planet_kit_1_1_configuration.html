Detailed Description

This is the class used to initialize PlanetKit. An instance of this class can be created using Configuration::Create().

◆ Create()

static ConfigurationPtr PlanetKit::Configuration::Create ( const wchar_t *  lpszBasePath,
const wchar_t *  lpszDatabasePath 
)
static

Create an instance of the Configuration class.

Parameters
lpszBasePathPath of PlanetKit binaries. You can use absolute or relative path.
String encoding must be UTF-16.
It can be an empty string like "", but it can't be nullptr.
lpszDatabasePathPath of PlanetKit database that includes some configuration options. You can use absolute or relative path.
String encoding must be UTF-16.
It can be an empty string like "", but it can't be nullptr.
Returns
  • If it successfully created an instance, it returns the pointer to the instance.
  • If it failed to create an instance, it returns nullptr.
Remarks
  • This API can fail when any parameters are nullptr.
  • If you put an empty string like L"", then the path will be the path of the current process.
