UserIdPtr  pPeerID,
const wchar_t *  szToken 
)
static

Creates an instance of the MakeCallParam class with an access token.

Parameters
pMyIDLocal user's ID and service ID
pPeerIDPeer's ID and service ID
szTokenAccess token string, which should be encoded in UTF-16 and null-terminated
Remarks
  • The parameter 'szToken' must have a string value.
  • This API can fail when the 'szToken' parameter is nullptr.
