◆ SetAppServerData() [2/2]

virtual void PlanetKit::ConferenceJoinParam::SetAppServerData ( const wchar_t *  szAppServerData)
pure virtual

Sets the app server data.

Parameters
szAppServerDataApp server data that is encoded in UTF-16.
Remarks
  • If the parameter is nullptr, the app server data stored in the class is cleared.
  • After converting szAppServerData to UTF-8, if it exceeds 4096 bytes including the null-termination character, the connection will fail.
  • In case of failure, the error code is PLNK_START_FAIL_REASON_TOO_LONG_APP_SERVER_DATA.
